Direct Acyclic Graph

I implemented a weighted direct acyclic graph utilizing C to find the single shortest path froma given source vertex to every other vertex in the graph. I also implemented depth first search in order to find the shortest path. Then checked the stack, since it was topologically sorted, to see if the given graph was a cycle
